# CodeMsg
CodeMsg is a similar to IPMsg for Visual Studio Code

## Features
* Send/Receive Message
* Extension with file attachment
* UTF-8 extension
* Chat history log
* Message Send/Receive encrypted extension(only support RSA-1024 and AES-256)
