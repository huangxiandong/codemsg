# CodeMsg
CodeMsg是一款于IPMsg兼容的的Visual Studio Code的扩展程序

## Features
* 发送接收消息
* 文件和文件夹发送和接收
* UTF-8扩展
* 聊天历史记录
* 接收和发送消息加密(目前只支持RSA-2048和AES-256, AES-256以及SHA1/SHA256)
